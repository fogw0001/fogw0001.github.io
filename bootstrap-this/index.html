Introduction to Bootstrap

Bootstrap is a useful toolkit for web development. It simplifies your code so that you can spend less time on the more difficult heaches that come with web development. Bootstrap has many useful features, such as JavaScript plugins, responsive grid systems, many useful prebuilt components, as well as sass variables and mixins.

Mobile First

Bootstrap uses a bunch of predefined classes to allow you to create all kinds of layouts using flexbox and grid. Following best practices, Bootstrap has a mobile first approach along with a twelve column system to fit with industry standards.

Themes

Bootstrap features tons of interesting themes to fit your design needs. These are component-based frameworks that work seemlessly with Bootstrap to make designing your website simple and effective.

Components

Bootstrap has many components available that are easy to use and help make your site more user friendly. Using these components requires no knowledge of JavaScript and are easy to add to your site.

How to Get Started

Getting started is made simple by Bootstrap. All you have to do is link to one stylesheet in the head of your html and add three lines of pre-made JavaScript to the body of your html. The stylesheet will allow you to use the css available to simplify your code and the JavaScript enables all the components. If you go to this link the stylesheet link and JavaScript is available for you. Alternatively, you can also find a starter template there, which has the css and JavaScript attached already, so you can just copy/paste the template and get started with Bootstrap!

Once you get set-up its as easy as adding the classes to your html, bootstrap will do the rest.

Buttons

Buttons are a common, but useful component. Buttons are good for indicating actions to the user, whether the action submits a form, links to a page, or activates a dialog. Buttons can be made into many different shapes and sizes to fit your design needs. Press the buttons below to show more examples of components.

Modals!

This is a modal. Modals are commonly used for lightboxes and user notifications, but they are flexible and can be used for many things. There are many stylistic choices for modals, you can make it so that clicking outside the modal box will close it or that you can only close it by hitting the X or close button. For longer content, you can make it so the modal will scroll without scrolling the page it is overlayed on.

Inside your modal you can use tooltips or popover windows to nest content inside of them. Also, you can use a grid system inside the modal window to align content to match your overall design. You can even create forms inside of your modals, the possibilites are endless!

Collapse is a useful tool for making certain content's visibility a toggle.
